# Mini Program UI Mockups

This folder contains Apple-inspired minimalist UI mockups for the tower-go mini program.

Screens:
- `01-login.svg` - login and brand entry
- `02-dashboard.svg` - workbench overview
- `03-inventory.svg` - inventory list and low-stock alerts
- `04-purchase.svg` - purchase order list
- `05-accounting.svg` - store accounting overview

Style direction:
- clean white surfaces
- subtle blue-gray gradients
- rounded cards
- restrained color accents
- strong information hierarchy
